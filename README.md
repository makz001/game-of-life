# Game of Life

![](./2023-sdl/life.gif)

## installation

```shell
$ cd 2023-sdl/
$ ./build.sh
```

## usage

```shell
$ ./a.out
```

## commands

- `q`: quit
- `space`: toggle mode [setup | running]

- **setup mode:**

    - `r`: reset game to the original state
    - `left click (on cell)` toggle cell state
